# binary-prefix

A golang module to parse Binary Prefixes for sizing Hard disks and Memory as
specified in https://en.wikipedia.org/wiki/Binary_prefix

## Tests

Running the tests:

```
go test .
```

Getting the lint report, just run:

```
go lint .
```
